## Application Overview

Vollna is a powerful Upwork automation tool that helps freelancers manage job proposals, auto-bidding, and notifications. Key features include:
- Advanced job filtering (30+ attributes)
- AI-powered proposal generation
- Auto-bidding system
- Real-time notifications
- Team collaboration
- Multiple integrations (Slack, Discord, Telegram, Zapier)

For a detailed Product Requirements Document, see: https://docs.google.com/document/d/10ywYjnCnUwLgH01GLaStNXkBgrwg5a6Sskln3lVvbjA/edit
